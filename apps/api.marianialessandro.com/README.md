# api.marianialessandro.com

Laravel API app for `api.marianialessandro.com`.

## Local Development

From the repository root:

```bash
npm run dev:api
```

Or from this app directory:

```bash
composer dev
```

The default API health endpoint is:

```text
GET /api/health
```

Run tests from the repository root:

```bash
npm run test:api
```
